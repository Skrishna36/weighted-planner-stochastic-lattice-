# weighted-planner-stochastic-lattice-
A Weighted Planar Stochastic Lattice is a specialized topological structure generated by partitioning a plane into smaller rectangular cells based on a specific, weight-dependent probability distribution.
Unlike standard random regular lattices, the WPSL exhibits unique scaling laws, fractal behaviors, and non-trivial network properties (such as power-law degree distributions). This project focuses on the precise algorithmic generation of these lattices and the subsequent analysis of their topological features, making it highly applicable to the study of non-equilibrium statistical mechanics, percolation theory, and complex networks.
